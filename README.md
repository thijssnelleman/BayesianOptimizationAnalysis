# BayesianOptimizationAnalysis

In this repo seperate files can be found to run BO on three modern day algorithms. Note that each BO for each algorithm is stand alone.

MLP/CNN files:
- graph_MLPCNN.py
- mlpcnn.py

Random Forest files:
- aml_random_forest.py

Decision Tree files:
- GuassianProcess.py
- RandomSearch.py
- RegressionTree.py
